# 🍔 Burger Bar – Luxus-Restaurant & Bar Website  

Eine stilvolle und moderne Website für ein gehobenes Burger-Restaurant und eine Bar.  
Erstellt mit **HTML, CSS und Vanilla JavaScript**, ganz klassisch ohne Frameworks.  

## 🚀 Live-Demo
